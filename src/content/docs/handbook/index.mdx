# Python ↔ TypeScript 手册

这份手册用于快速对照两种语言的高频语法和工程概念。

## 常用映射

- 变量声明：Python 直接赋值；TypeScript 常用 `const` / `let`
- 类型标注：Python 使用 `x: int`；TypeScript 使用 `x: number`
- 模块导入：Python `import`；TypeScript `import` + ESM
- 异步模型：Python `asyncio`；TypeScript `Promise` + `async/await`

2. 遇到语法差异时回到手册快速查阅
