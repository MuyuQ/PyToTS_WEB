# Welcome to Python to TypeScript Migration Guide

This interactive learning path helps Python developers master TypeScript through structured lessons, side-by-side code comparisons, and hands-on exercises.

## Why TypeScript?

TypeScript adds static typing to JavaScript, bringing many concepts familiar to Python developers while enabling better tooling, IDE support, and error catching at compile time.

## Learning Paths

Choose your learning path based on your experience level:

- **Beginner**: TypeScript fundamentals for Python developers
- **Intermediate**: Advanced types and patterns
- **Advanced**: Metaprogramming and design patterns

## Key Differences

| Python | TypeScript |
|--------|------------|
| Dynamic typing | Static typing |
| `def` for functions | `function` or arrow functions |
| Indentation-based blocks | Brace-based blocks |
| `list`, `dict` | `Array`, `Record`, `Map` |
