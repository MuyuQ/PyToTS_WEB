# Type System Comparison

This guide compares Python's type system with TypeScript's approach to typing.

## Basic Types

### Numbers

**Python:**
```python
x: int = 10
y: float = 3.14
```

**TypeScript:**
```typescript
const x: number = 10;
const y: number = 3.14;
```

TypeScript uses `number` for both integers and floats, similar to JavaScript.

### Strings

**Python:**
```python
name: str = "Alice"
```

**TypeScript:**
```typescript
const name: string = "Alice";
```

### Booleans

**Python:**
```python
is_active: bool = True
```

**TypeScript:**
```typescript
const isActive: boolean = true;
```

## Collections

### Lists/Arrays

**Python:**
```python
numbers: list[int] = [1, 2, 3]
names: list[str] = ["Alice", "Bob"]
```

**TypeScript:**
```typescript
const numbers: number[] = [1, 2, 3];
const names: string[] = ["Alice", "Bob"];
// Or using generic syntax:
const numbers2: Array<number> = [1, 2, 3];
```

### Dictionaries/Objects

**Python:**
```python
person: dict[str, str] = {"name": "Alice", "city": "NYC"}
```

**TypeScript:**
```typescript
const person: Record<string, string> = { name: "Alice", city: "NYC" };
// Or using interface:
interface Person {
  name: string;
  city: string;
}
const person2: Person = { name: "Alice", city: "NYC" };
```

## Union Types

**Python:**
```python
from typing import Union

value: Union[int, str] = 42
# Or with Python 3.10+:
value2: int | str = "hello"
```

**TypeScript:**
```typescript
let value: number | string = 42;
value = "hello";
```

## Optional Types

**Python:**
```python
from typing import Optional

def greet(name: Optional[str] = None) -> str:
    if name:
        return f"Hello, {name}!"
    return "Hello!"
```

**TypeScript:**
```typescript
function greet(name?: string): string {
    if (name) {
        return `Hello, ${name}!`;
    }
    return "Hello!";
}

// Or explicitly:
function greet2(name: string | undefined): string {
    // ...
}
```
