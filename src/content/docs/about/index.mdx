# About & Contributing

这是一个面向 Python 开发者的 TypeScript 学习站点，采用内容优先的静态文档架构。

## 项目目标

- 用并排对照降低语言迁移成本
- 用算法题提升实战与面试能力
- 用可维护的静态站点保证长期可迭代

## 参与贡献

- 阅读 `CONTRIBUTING.md`
- 保持「一个可验证功能切片一个提交」
- 提交前运行 `npm run check`
