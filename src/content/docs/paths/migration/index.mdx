## 学习建议

1. 按顺序学习四节核心课，先建立类型心智再进入异步与模块。
2. 每节课都把 Python 示例手动改写为 TypeScript，强化迁移肌肉记忆。
3. 学完后立即进入 [算法练习](/algorithms/) 做双语实战。
