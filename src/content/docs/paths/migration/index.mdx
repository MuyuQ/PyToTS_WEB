## 学习建议

1. 按顺序学习核心课程，先建立类型心智再进入异步与模块。
2. 每节课都把 Python 示例手动改写为 TypeScript，强化迁移肌肉记忆。
3. 重点关注错误处理和字符串处理，这是日常开发的高频操作。
4. 学完后立即进入 [算法练习](/algorithms/) 做双语实战。
