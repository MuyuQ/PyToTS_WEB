## 学习建议

1. **按顺序学习**：每节课都建立在前面的基础上
2. **动手实践**：在 [Playground](https://www.typescriptlang.org/play) 上练习
3. **对比记忆**：刻意对比 Python 和 TypeScript 的语法差异
4. **完成练习**：每节课后的练习要认真完成

## 完成后你能

- ✓ 编写基本的 TypeScript 程序
- ✓ 理解类型系统的基本概念
- ✓ 使用函数、类、接口组织代码
- ✓ 为后续迁移路径打下坚实基础
