## 学习建议

1. **先掌握基础**：确保已完成 [基础路径](/paths/foundation/) 和 [迁移路径](/paths/migration/)
2. **动手实践**：每个设计模式都要自己实现一遍
3. **对比思考**：对比 Python 和 TypeScript 的实现差异
4. **准备面试**：重点关注泛型和设计模式的面试题

## 完成后你能

- ✓ 编写类型安全且可复用的泛型代码
- ✓ 精确控制类型收窄和类型守卫
- ✓ 使用装饰器增强代码功能
- ✓ 在项目中应用常见设计模式
- ✓ 自信地回答高级 TypeScript 面试题

## 推荐实践

完成进阶路径后，建议：

1. **贡献开源**：参与 TypeScript 项目的开源贡献
2. **构建项目**：用 TypeScript 构建一个完整的应用
3. **刷算法题**：在 [算法练习](/algorithms/) 中挑战 Hard 级别题目
4. **准备面试**：复习所有路径的核心概念
