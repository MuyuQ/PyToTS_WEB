## 📚 学习顺序建议

### 第一条：基础路径（必学）

适合所有人，建立 TypeScript 基础：

1. [变量与数据类型](/paths/foundation/variables) - TypeScript 的类型系统基础
2. [函数基础](/paths/foundation/functions-basics) - 函数定义与类型注解
3. [控制流](/paths/foundation/control-flow) - 条件语句与循环
4. [数据结构](/paths/foundation/data-structures) - Array、Object、Map、Set
5. [类与对象](/paths/foundation/classes-objects) - 面向对象编程

### 第二条：迁移路径（有 Python 经验必学）

系统对比 Python 与 TypeScript 的差异：

1. [类型系统对比](/paths/migration/types) - 理解类型检查的核心差异
2. [函数与参数](/paths/migration/functions) - 参数传递与返回值差异
3. [异步编程](/paths/migration/async) - Promise vs asyncio
4. [模块与包管理](/paths/migration/modules) - ES6 模块 vs Python 导入

### 第三条：进阶路径（深入学习）

工程实践与面试准备：

1. [泛型](/paths/advanced/generics) - 可复用的类型抽象
2. [类型守卫](/paths/advanced/type-guards) - 类型收窄与类型推断
3. [装饰器](/paths/advanced/decorators) - 元编程与 AOP
4. [设计模式](/paths/advanced/design-patterns) - 常见设计模式的 TS 实现
