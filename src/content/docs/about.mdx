# 关于与贡献

## 项目介绍

本项目旨在帮助 Python 开发者学习 TypeScript，通过：

- **直接对比**：Python 和 TypeScript 代码并排展示
- **算法实战**：双语实现经典算法题
- **面试准备**：覆盖常见面试题型

## 如何贡献

### 贡献内容

1. Fork 本仓库
2. 创建新分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "feat: your feature"`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

### 内容规范

- 每个课程需遵循 7 段式模板
- 算法页面需遵循 8 段式模板
- 代码需包含双语实现和注释
