## 练习建议

1. **先学习再测验**：建议先完成对应的 [学习路径](/paths/) 内容
2. **边做边对比**：在算法练习中，刻意对比 Python 和 TypeScript 的实现差异
3. **定期自测**：使用测验功能检验学习成果
4. **总结规律**：建立自己的「Python ↔ TypeScript 语法对照表」

## 自测清单

完成以下检查项，确保你已准备好进行 TypeScript 开发：

- [ ] 我能解释同一段逻辑在 Python 和 TypeScript 的类型差异
- [ ] 我能为函数参数与返回值写出清晰类型
- [ ] 我能读懂并改写 Promise/async 代码
- [ ] 我能熟练使用 TypeScript 的类型推断和类型断言
- [ ] 我能处理可选值和默认值的情况
- [ ] 我能理解 TypeScript 的接口和类型别名
