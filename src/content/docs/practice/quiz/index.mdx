## 如何有效利用测验

1. **先学习再测验**：建议先完成对应的 [学习路径](/paths/) 内容
2. **分析错误**：每道题都有详细解释，理解为什么错比答对更重要
3. **多次练习**：可以重复进行测验，直到掌握所有概念
4. **对比记忆**：重点关注 Python 和 TypeScript 的语法差异

## 学习建议

- **类型系统**：这是 Python 开发者学习 TypeScript 最大的障碍，建议重点练习
- **异步编程**：Promise 和 async/await 的概念需要理解其执行机制
- **函数差异**：箭头函数、默认参数、剩余参数等语法需要熟练
- **泛型**：TypeScript 泛型比 Python 更强大，需要多加练习
- **工具类型**：Partial、Pick、Omit 等工具类型能极大提高开发效率
