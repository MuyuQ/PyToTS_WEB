## 问题描述

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s`，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合
2. 左括号必须以正确的顺序闭合

## 思路分析

使用栈来匹配括号：
- 遇到左括号就压入栈
- 遇到右括号就检查栈顶是否匹配
- 最后栈应该为空

## 复杂度分析

- **时间复杂度**：O(n)，只需遍历一次字符串
- **空间复杂度**：O(n)，栈最多存储 n 个字符

## Python 实现

```python
def is_valid(s: str) -> bool:
    """
    有效的括号 - Python 实现
    
    Args:
        s: 括号字符串
    
    Returns:
        括号是否有效
    """
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping:
            # 右括号，检查栈顶
            if not stack or stack[-1] != mapping[char]:
                return False
            stack.pop()
        else:
            # 左括号，入栈
            stack.append(char)
    
    return len(stack) == 0


# 测试用例
print(is_valid("()"))      # True
print(is_valid("()[]{}"))  # True
print(is_valid("(]"))      # False
print(is_valid("([)]"))    # False
```

## TypeScript 实现

```typescript
/**
 * 有效的括号 - TypeScript 实现
 * @param s 括号字符串
 * @returns 括号是否有效
 */
function isValid(s: string): boolean {
    const stack: string[] = [];
    const mapping: Record<string, string> = { ')': '(', '}': '{', ']': '[' };
    
    for (const char of s) {
        if (char in mapping) {
            // 右括号，检查栈顶
            if (stack.length === 0 || stack[stack.length - 1] !== mapping[char]) {
                return false;
            }
            stack.pop();
        } else {
            // 左括号，入栈
            stack.push(char);
        }
    }
    
    return stack.length === 0;
}

// 测试用例
console.log(isValid("()"));      // true
console.log(isValid("()[]{}"));  // true
console.log(isValid("(]"));      // false
console.log(isValid("([)]"));    // false
```

## 面试变体

1. **包含其他字符**：字符串中包含非括号字符，忽略它们
2. **最少添加括号**：计算让字符串有效需要添加的最少括号数
3. **最长有效子串**：找出最长的有效括号子串的长度

## 常见错误

1. 忘记检查栈为空的情况就取栈顶
2. 混淆左右括号的映射关系
3. 最后忘记检查栈是否为空
