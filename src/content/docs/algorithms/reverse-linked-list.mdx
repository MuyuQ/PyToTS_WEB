## 📝 问题描述

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

```
输入: 1 → 2 → 3 → 4 → 5 → None
输出: 5 → 4 → 3 → 2 → 1 → None
```

## 💡 思路引导

**第一步：理解问题**

链表反转，就是要把箭头方向全部反过来：

- 原来：1 → 2 → 3 → 4 → 5 → None
- 目标：None ← 1 ← 2 ← 3 ← 4 ← 5

**第二步：关键观察**

每个节点有个 `next` 指针，指向下一个节点。要反转链表，就要把每个节点的 `next` 指针"调头"，指向前一个节点。

但问题来了：如果我先把节点 1 的 `next` 改成 None，那节点 2 去哪找？链表断了！

**第三步：需要哪些变量？**

为了避免"链表断了"的问题，需要三个指针：

| 变量        | 作用                       | 初始值 |
| ----------- | -------------------------- | ------ |
| `prev`      | 前一个节点（反转后指向它） | `None` |
| `curr`      | 当前正在处理的节点         | `head` |
| `next_temp` | 暂存下一个节点（防止断链） | -      |

**第四步：每一步做什么？**

对于每个节点，执行四步操作：

1. **保存后路**：`next_temp = curr.next`（先记住下一个是谁，不然改完就找不到了）
2. **反转指针**：`curr.next = prev`（把箭头指向前一个）
3. **前进准备**：`prev = curr`（前一个节点变成当前节点）
4. **移动指针**：`curr = next_temp`（处理下一个节点）

**第五步：什么时候结束？**

当 `curr` 变成 `None` 时，说明已经处理完所有节点。此时 `prev` 就是新的头节点。

## 🎯 生活化类比

反转链表就像**翻书**：

想象你手里有一叠连在一起的书签，每张书签用绳子连着下一张：

```
[书签1] —绳子→ [书签2] —绳子→ [书签3] —绳子→ [书签4]
```

现在要把绳子方向反过来，但你只有两只手，不能同时抓住所有书签。

**解决办法**：

1. 左手拿"上一张"（prev，一开始是空的）
2. 右手拿"当前这张"（curr）
3. 先用备用夹子夹住"下一张"的位置（next_temp）
4. 把当前这张的绳子解下来，绑到左手那张上
5. 左右手各往前挪一张
6. 重复，直到右手拿的是空气（None）

最后左手拿的就是新的第一张！

## ⚠️ 常见错误分析

### 错误 1：忘记保存下一个节点

```python
# ❌ 错误写法
while curr:
    curr.next = prev    # 先改了指针
    prev = curr
    curr = curr.next    # 这里 curr.next 已经是 prev 了！
```

**后果**：链表断了，`curr` 指向了前一个节点而不是下一个节点，导致死循环或提前退出。

**正确写法**：

```python
# ✅ 正确写法
while curr:
    next_temp = curr.next   # 先保存下一个节点
    curr.next = prev        # 再反转
    prev = curr
    curr = next_temp        # 用保存的值
```

### 错误 2：返回错误的节点

```python
# ❌ 错误写法
return curr    # 循环结束时 curr 是 None！
```

**原因**：循环条件是 `while curr`，当 `curr` 变成 `None` 时才退出。此时 `curr` 是空指针，不是头节点。

**正确写法**：

```python
# ✅ 正确写法
return prev    # prev 指向反转后的最后一个节点，也就是新头
```

### 错误 3：递归忘记返回新头

```python
# ❌ 错误写法（递归版本）
def reverse_list(head):
    if not head or not head.next:
        return head
    reverse_list(head.next)    # 忘记接收返回值
    head.next.next = head
    head.next = None
    # 忘记返回新头！
```

**后果**：外层调用拿不到新头节点，返回 `None`。

**正确写法**：

```python
# ✅ 正确写法（递归版本）
def reverse_list(head):
    if not head or not head.next:
        return head
    new_head = reverse_list(head.next)  # 接收返回值
    head.next.next = head
    head.next = None
    return new_head                      # 返回新头
```

### 📝 记忆口诀

反转链表三步走：

1. **先存后**：`next_temp = curr.next`
2. **再反指**：`curr.next = prev`
3. **往前移**：`prev = curr, curr = next_temp`

最后返回 `prev`，不是 `curr`！
