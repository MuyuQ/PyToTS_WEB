## 问题描述

给定一个整数数组 `nums`，数组中的元素**互不相同**。返回该数组所有可能的子集（幂集）。

**要求**：解集不能包含重复的子集，可以按任意顺序返回。

**示例**：

- 输入：`nums = [1, 2, 3]`
- 输出：`[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]`

## 思路分析

### 方法一：回溯

子集问题本质是在每个元素前做**选择**：选或不选。回溯法通过递归遍历决策树：

1. 从第一个元素开始，依次考虑每个位置
2. 对于当前位置，有两种选择：加入子集 或 不加入
3. 递归处理下一个位置
4. 当遍历完所有元素时，收集当前路径作为一个子集

关键点：子集问题不需要终止条件判断，每个递归状态都是一个有效子集。

### 方法二：迭代

从空集开始，依次考虑每个元素：

1. 初始时结果集只包含空集 `[[]]`
2. 对于每个新元素，将现有所有子集复制一份，并加入该元素
3. 将新子集合并到结果集中

例如 `[1, 2]`：

- 初始：`[[]]`
- 加入 1：`[[], [1]]`
- 加入 2：`[[], [1], [2], [1, 2]]`

### 方法三：位运算

n 个元素的子集数恰为 2^n，可以用 n 位二进制数表示：

- 第 i 位为 1 表示选择 nums[i]
- 第 i 位为 0 表示不选择

遍历 0 到 2^n - 1，每个数字对应一个子集。

## 复杂度分析

- **时间复杂度**：O(n × 2^n)，共 2^n 个子集，每个子集平均 O(n) 时间构造
- **空间复杂度**：O(n)，递归栈深度或临时数组长度（不计结果集）

    ```python
    # 子集 (幂集) - 三种实现方法
    # 时间复杂度: O(n × 2^n), 空间复杂度: O(n)
    # n 为数组长度, 共 2^n 个子集

    from typing import List

    # === 方法一: 回溯法 ===

    def subsets_backtrack(nums: List[int]) -> List[List[int]]: # result: 存储所有子集
    result = [] # n: 数组长度
    n = len(nums)

        def backtrack(start: int, path: List[int]) -> None:
            # start: 当前考虑的起始位置 (避免重复选择)
            # path: 当前已选择的元素
            # 每个递归状态都是一个有效子集, 立即收集
            result.append(path[:])  # 注意: 必须拷贝, 否则后续 pop 会影响已保存的结果

            # 从 start 开始遍历剩余元素
            for i in range(start, n):
                # 选择: 将 nums[i] 加入子集
                path.append(nums[i])
                # 递归: 从 i+1 开始继续选择 (不能回头, 避免重复)
                backtrack(i + 1, path)
                # 撤销: 移除最后加入的元素 (回溯)
                path.pop()

        # 从位置 0、空路径开始回溯
        backtrack(0, [])
        return result

    # === 方法二: 迭代法 ===

    def subsets_iterative(nums: List[int]) -> List[List[int]]: # result: 初始只包含空集
    result = [[]]

        # 依次考虑每个元素
        for num in nums:
            # 对每个现有子集, 添加当前元素生成新子集
            new_subsets = []
            for subset in result:
                # 拷贝原子集并添加当前元素
                new_subsets.append(subset + [num])
            # 将新生成的子集合并到结果中
            result.extend(new_subsets)

        return result

    # === 方法三: 位运算法 ===

    def subsets_bitmask(nums: List[int]) -> List[List[int]]: # n: 数组长度
    n = len(nums) # result: 存储所有子集
    result = []

        # 遍历所有可能的掩码: 0 到 2^n - 1
        # 每个掩码的二进制位表示对应元素是否被选中
        for mask in range(1 << n):  # 1 << n 等于 2^n
            subset = []
            # 检查掩码的每一位
            for i in range(n):
                # 如果第 i 位为 1, 说明选择 nums[i]
                # mask & (1 << i): 将 1 左移 i 位后与 mask 做按位与
                if mask & (1 << i):
                    subset.append(nums[i])
            # 将生成的子集加入结果
            result.append(subset)

        return result

    # 测试用例

    nums = [1, 2, 3]

    # 三种方法都应输出 8 个子集 (2^3 = 8)

    print("回溯法:", subsets_backtrack(nums))
    print("迭代法:", subsets_iterative(nums))
    print("位运算:", subsets_bitmask(nums))

    # 验证结果数量

    print(f"子集总数: {len(subsets_backtrack(nums))}") # 输出: 8 = 2^3
    ```

## 面试变体

1. **子集 II**：数组包含重复元素，返回不重复的子集（需要先排序，跳过重复元素）
2. **组合**：从 n 个数中选 k 个的组合（在子集基础上限制长度）
3. **排列**：全排列（交换法或标记法）
4. **字母大小写全排列**：字符串中字母可变大写或小写
5. **递增子序列**：找出所有递增子序列

## 常见错误

1. **忘记拷贝**：`result.append(path)` 而不是 `result.append(path[:])`，导致结果全为空
2. **起始位置错误**：`backtrack(i, path)` 而不是 `backtrack(i + 1, path)`，导致重复选择
3. **位运算优先级**：`mask & 1 << i` 应写成 `mask & (1 << i)`
4. **迭代法数组引用**：直接修改正在遍历的数组，应使用临时数组收集新子集
5. **重复元素未处理**：子集 II 中忘记排序和跳过重复元素
