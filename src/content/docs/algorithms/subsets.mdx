## 问题描述

给定一个整数数组 `nums`，数组中的元素**互不相同**。返回该数组所有可能的子集（幂集）。

**要求**：解集不能包含重复的子集，可以按任意顺序返回。

**示例**：

- 输入：`nums = [1, 2, 3]`
- 输出：`[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3]]`

## 思路分析

### 方法一：回溯

子集问题本质是在每个元素前做**选择**：选或不选。回溯法通过递归遍历决策树：

1. 从第一个元素开始，依次考虑每个位置
2. 对于当前位置，有两种选择：加入子集 或 不加入
3. 递归处理下一个位置
4. 当遍历完所有元素时，收集当前路径作为一个子集

关键点：子集问题不需要终止条件判断，每个递归状态都是一个有效子集。

### 方法二：迭代

从空集开始，依次考虑每个元素：

1. 初始时结果集只包含空集 `[[]]`
2. 对于每个新元素，将现有所有子集复制一份，并加入该元素
3. 将新子集合并到结果集中

例如 `[1, 2]`：

- 初始：`[[]]`
- 加入 1：`[[], [1]]`
- 加入 2：`[[], [1], [2], [1, 2]]`

### 方法三：位运算

n 个元素的子集数恰为 2^n，可以用 n 位二进制数表示：

- 第 i 位为 1 表示选择 nums[i]
- 第 i 位为 0 表示不选择

遍历 0 到 2^n - 1，每个数字对应一个子集。

## 复杂度分析

- **时间复杂度**：O(n × 2^n)，共 2^n 个子集，每个子集平均 O(n) 时间构造
- **空间复杂度**：O(n)，递归栈深度或临时数组长度（不计结果集）

## Python 实现

```python
from typing import List

def subsets_backtrack(nums: List[int]) -> List[List[int]]:
    """
    子集 - 回溯实现

    Args:
        nums: 整数数组（元素互不相同）

    Returns:
        所有可能的子集
    """
    result = []
    n = len(nums)

    def backtrack(start: int, path: List[int]) -> None:
        """
        回溯函数

        Args:
            start: 当前选择的起始位置
            path: 当前已选择的元素
        """
        # 每个状态都是一个有效子集
        result.append(path[:])  # 注意：需要拷贝

        # 从 start 开始遍历，避免重复
        for i in range(start, n):
            # 选择当前元素
            path.append(nums[i])
            # 递归处理后续元素
            backtrack(i + 1, path)
            # 撤销选择（回溯）
            path.pop()

    backtrack(0, [])
    return result


def subsets_iterative(nums: List[int]) -> List[List[int]]:
    """
    子集 - 迭代实现

    Args:
        nums: 整数数组（元素互不相同）

    Returns:
        所有可能的子集
    """
    result = [[]]  # 初始只有空集

    for num in nums:
        # 对每个现有子集，添加当前元素生成新子集
        new_subsets = []
        for subset in result:
            # 拷贝并添加当前元素
            new_subsets.append(subset + [num])
        # 合并新子集
        result.extend(new_subsets)

    return result


def subsets_bitmask(nums: List[int]) -> List[List[int]]:
    """
    子集 - 位运算实现

    Args:
        nums: 整数数组（元素互不相同）

    Returns:
        所有可能的子集
    """
    n = len(nums)
    result = []

    # 遍历所有可能的掩码（0 到 2^n - 1）
    for mask in range(1 << n):  # 2^n 种情况
        subset = []
        for i in range(n):
            # 检查第 i 位是否为 1
            if mask & (1 << i):
                subset.append(nums[i])
        result.append(subset)

    return result


# 测试用例
nums = [1, 2, 3]

print("回溯法:", subsets_backtrack(nums))
print("迭代法:", subsets_iterative(nums))
print("位运算:", subsets_bitmask(nums))

# 验证结果数量
print(f"子集总数: {len(subsets_backtrack(nums))}")  # 8 = 2^3
```

## TypeScript 实现

```typescript
/**
 * 子集 - 回溯实现
 * @param nums 整数数组（元素互不相同）
 * @returns 所有可能的子集
 */
function subsetsBacktrack(nums: number[]): number[][] {
  const result: number[][] = [];
  const n = nums.length;

  /**
   * 回溯函数
   * @param start 当前选择的起始位置
   * @param path 当前已选择的元素
   */
  function backtrack(start: number, path: number[]): void {
    // 每个状态都是一个有效子集
    result.push([...path]); // 注意：需要拷贝

    // 从 start 开始遍历，避免重复
    for (let i = start; i < n; i++) {
      // 选择当前元素
      path.push(nums[i]);
      // 递归处理后续元素
      backtrack(i + 1, path);
      // 撤销选择（回溯）
      path.pop();
    }
  }

  backtrack(0, []);
  return result;
}

/**
 * 子集 - 迭代实现
 * @param nums 整数数组（元素互不相同）
 * @returns 所有可能的子集
 */
function subsetsIterative(nums: number[]): number[][] {
  const result: number[][] = [[]]; // 初始只有空集

  for (const num of nums) {
    // 对每个现有子集，添加当前元素生成新子集
    const newSubsets: number[][] = [];
    for (const subset of result) {
      // 拷贝并添加当前元素
      newSubsets.push([...subset, num]);
    }
    // 合并新子集
    result.push(...newSubsets);
  }

  return result;
}

/**
 * 子集 - 位运算实现
 * @param nums 整数数组（元素互不相同）
 * @returns 所有可能的子集
 */
function subsetsBitmask(nums: number[]): number[][] {
  const n = nums.length;
  const result: number[][] = [];

  // 遍历所有可能的掩码（0 到 2^n - 1）
  for (let mask = 0; mask < 1 << n; mask++) {
    // 2^n 种情况
    const subset: number[] = [];
    for (let i = 0; i < n; i++) {
      // 检查第 i 位是否为 1
      if (mask & (1 << i)) {
        subset.push(nums[i]);
      }
    }
    result.push(subset);
  }

  return result;
}

// 测试用例
const nums = [1, 2, 3];

console.log("回溯法:", subsetsBacktrack(nums));
console.log("迭代法:", subsetsIterative(nums));
console.log("位运算:", subsetsBitmask(nums));

// 验证结果数量
console.log(`子集总数: ${subsetsBacktrack(nums).length}`); // 8 = 2^3
```

## 面试变体

1. **子集 II**：数组包含重复元素，返回不重复的子集（需要先排序，跳过重复元素）
2. **组合**：从 n 个数中选 k 个的组合（在子集基础上限制长度）
3. **排列**：全排列（交换法或标记法）
4. **字母大小写全排列**：字符串中字母可变大写或小写
5. **递增子序列**：找出所有递增子序列

## 常见错误

1. **忘记拷贝**：`result.append(path)` 而不是 `result.append(path[:])`，导致结果全为空
2. **起始位置错误**：`backtrack(i, path)` 而不是 `backtrack(i + 1, path)`，导致重复选择
3. **位运算优先级**：`mask & 1 << i` 应写成 `mask & (1 << i)`
4. **迭代法数组引用**：直接修改正在遍历的数组，应使用临时数组收集新子集
5. **重复元素未处理**：子集 II 中忘记排序和跳过重复元素
