## 问题描述

给定一个 `m x n` 的二维字符网格 `board` 和一个字符串 `word`。如果 `word` 存在于网格中，返回 `true`；否则，返回 `false`。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中**相邻**单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母不允许被重复使用。

## 思路分析

回溯 + DFS：

1. 从每个单元格出发，尝试匹配单词的第一个字符
2. 如果匹配，向四个方向递归搜索下一个字符
3. 使用访问标记避免重复使用单元格
4. 回溯时恢复状态

## 复杂度分析

- **时间复杂度**：O(m _ n _ 4^l)，l 是单词长度，每个单元格出发，最多4个方向
- **空间复杂度**：O(l)，递归栈深度

                 dfs(i, j + 1, k + 1) or  # 向右

## 面试变体

1. **单词搜索 II**：给定多个单词，找出所有存在的单词
2. **前缀树优化**：使用 Trie 优化多个单词的搜索
3. **最短路径**：找到构成单词的最短路径

## 常见错误

1. 忘记回溯恢复状态
2. 边界检查条件写错
3. 递归终止条件写错（应该是 k == len-1 不是 k == len）
