## 问题描述

给你一个由 `'1'`（陆地）和 `'0'`（水）组成的的二维网格，请你计算网格中岛屿的数量。

岛屿总是被水包围，并且每座岛屿只能由水平方向或竖直方向上相邻的陆地连接形成。

## 思路分析

DFS/BFS：

- 遍历每个单元格
- 遇到陆地 `'1'`，岛屿数量 +1
- 使用 DFS/BFS 将所有相连的陆地标记为已访问（改为 `'0'` 或 `'2'`）
- 继续遍历

## 复杂度分析

- **时间复杂度**：O(m \* n)，每个单元格最多访问一次
- **空间复杂度**：O(m \* n)，递归栈或队列的最坏情况

## 面试变体

1. **岛屿的最大面积**：求最大岛屿的面积
2. **被围绕的区域**：将被 'X' 围绕的 'O' 填充为 'X'
3. **太平洋大西洋水流**：求能流到两个大洋的单元格

## 常见错误

1. 忘记边界检查
2. 标记访问状态错误
3. 混淆行和列的索引
