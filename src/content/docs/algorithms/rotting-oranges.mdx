## 问题描述

在给定的 `m x n` 网格 `grid` 中，每个单元格可以有以下三个值之一：
- `0` 代表空单元格
- `1` 代表新鲜橘子
- `2` 代表腐烂橘子

每分钟，任何与腐烂橘子相邻（上下左右）的新鲜橘子都会腐烂。

返回直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能，返回 `-1`。

## 思路分析

多源 BFS：
1. 初始化：将所有腐烂橘子加入队列，统计新鲜橘子数量
2. BFS：每分钟处理当前所有腐烂橘子，使相邻新鲜橘子腐烂
3. 终止：队列为空时，检查是否还有新鲜橘子

## 复杂度分析

- **时间复杂度**：O(m * n)，每个单元格最多访问一次
- **空间复杂度**：O(m * n)，队列的最坏情况

## Python 实现

```python
from collections import deque

def oranges_rotting(grid: list[list[int]]) -> int:
    """
    腐烂的橘子 - Python 实现 (多源BFS)
    
    Args:
        grid: 橘子网格
    
    Returns:
        最短时间，如果不可能则返回-1
    """
    if not grid or not grid[0]:
        return -1
    
    m, n = len(grid), len(grid[0])
    queue = deque()
    fresh = 0
    
    # 初始化：找到所有腐烂橘子，统计新鲜橘子
    for i in range(m):
        for j in range(n):
            if grid[i][j] == 2:
                queue.append((i, j, 0))  # (行, 列, 时间)
            elif grid[i][j] == 1:
                fresh += 1
    
    # 如果没有新鲜橘子，返回0
    if fresh == 0:
        return 0
    
    # 如果没有腐烂橘子，无法传播
    if not queue:
        return -1
    
    minutes = 0
    directions = [(0, 1), (0, -1), (1, 0), (-1, 0)]
    
    while queue:
        i, j, time = queue.popleft()
        minutes = max(minutes, time)
        
        for di, dj in directions:
            ni, nj = i + di, j + dj
            
            # 边界检查和新鲜橘子检查
            if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == 1:
                grid[ni][nj] = 2  # 腐烂
                fresh -= 1
                queue.append((ni, nj, time + 1))
    
    return -1 if fresh > 0 else minutes


# 测试用例
grid1 = [[2, 1, 1], [1, 1, 0], [0, 1, 1]]
print(oranges_rotting(grid1))  # 4

grid2 = [[2, 1, 1], [0, 1, 1], [1, 0, 2]]
print(oranges_rotting(grid2))  # -1

grid3 = [[0, 2]]
print(oranges_rotting(grid3))  # 0
```

## TypeScript 实现

```typescript
/**
 * 腐烂的橘子 - TypeScript 实现 (多源BFS)
 * @param grid 橘子网格
 * @returns 最短时间，如果不可能则返回-1
 */
function orangesRotting(grid: number[][]): number {
    if (grid.length === 0 || grid[0].length === 0) {
        return -1;
    }
    
    const m = grid.length;
    const n = grid[0].length;
    const queue: [number, number, number][] = [];  // [行, 列, 时间]
    let fresh = 0;
    
    // 初始化：找到所有腐烂橘子，统计新鲜橘子
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            if (grid[i][j] === 2) {
                queue.push([i, j, 0]);
            } else if (grid[i][j] === 1) {
                fresh++;
            }
        }
    }
    
    // 如果没有新鲜橘子，返回0
    if (fresh === 0) {
        return 0;
    }
    
    // 如果没有腐烂橘子，无法传播
    if (queue.length === 0) {
        return -1;
    }
    
    let minutes = 0;
    const directions = [[0, 1], [0, -1], [1, 0], [-1, 0]];
    
    while (queue.length > 0) {
        const [i, j, time] = queue.shift()!;
        minutes = Math.max(minutes, time);
        
        for (const [di, dj] of directions) {
            const ni = i + di;
            const nj = j + dj;
            
            // 边界检查和新鲜橘子检查
            if (ni >= 0 && ni < m && nj >= 0 && nj < n && grid[ni][nj] === 1) {
                grid[ni][nj] = 2;  // 腐烂
                fresh--;
                queue.push([ni, nj, time + 1]);
            }
        }
    }
    
    return fresh > 0 ? -1 : minutes;
}

// 测试用例
const grid1 = [[2, 1, 1], [1, 1, 0], [0, 1, 1]];
console.log(orangesRotting(grid1));  // 4

const grid2 = [[2, 1, 1], [0, 1, 1], [1, 0, 2]];
console.log(orangesRotting(grid2));  // -1

const grid3 = [[0, 2]];
console.log(orangesRotting(grid3));  // 0
```

## 面试变体

1. **墙阻隔**：某些单元格是墙，橘子不能穿过
2. **多个传播源**：不同类型的腐烂源传播速度不同
3. **最大新鲜时间**：求每个新鲜橘子被腐烂的最短时间

## 常见错误

1. 忘记处理没有新鲜橘子或没有腐烂橘子的边界情况
2. BFS时忘记更新时间
3. 最后忘记检查是否还有新鲜橘子
