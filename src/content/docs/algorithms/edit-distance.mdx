## 问题描述

给你两个单词 `word1` 和 `word2`，请返回将 `word1` 转换成 `word2` 所使用的最少操作数。

你可以对一个单词进行如下三种操作：
- 插入一个字符
- 删除一个字符
- 替换一个字符

## 思路分析

动态规划：
- `dp[i][j]`：`word1[0..i-1]` 转换成 `word2[0..j-1]` 的最少操作数
- 状态转移：
  - 如果 `word1[i-1] == word2[j-1]`：`dp[i][j] = dp[i-1][j-1]`
  - 否则：`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`
    - `dp[i-1][j]` + 1：删除 `word1[i-1]`
    - `dp[i][j-1]` + 1：插入 `word2[j-1]`
    - `dp[i-1][j-1]` + 1：替换 `word1[i-1]` 为 `word2[j-1]`

## 复杂度分析

- **时间复杂度**：O(m * n)，m 和 n 是两个单词长度
- **空间复杂度**：O(n)，可以优化为一维数组

## Python 实现

```python
def min_distance(word1: str, word2: str) -> int:
    """
    编辑距离 - Python 实现
    
    Args:
        word1: 单词1
        word2: 单词2
    
    Returns:
        最少操作数
    """
    m, n = len(word1), len(word2)
    
    # dp[j] 表示 word1[0..i-1] 转换成 word2[0..j-1] 的最少操作数
    dp = list(range(n + 1))
    
    for i in range(1, m + 1):
        prev = dp[0]  # 保存 dp[i-1][j-1]
        dp[0] = i     # 第一列
        
        for j in range(1, n + 1):
            temp = dp[j]  # 保存 dp[i-1][j]
            
            if word1[i - 1] == word2[j - 1]:
                dp[j] = prev
            else:
                dp[j] = min(dp[j], dp[j - 1], prev) + 1
            
            prev = temp
    
    return dp[n]


# 测试用例
print(min_distance("horse", "ros"))     # 3 (horse -> rorse -> rose -> ros)
print(min_distance("intention", "execution"))  # 5
print(min_distance("", ""))             # 0
print(min_distance("a", ""))            # 1
```

## TypeScript 实现

```typescript
/**
 * 编辑距离 - TypeScript 实现
 * @param word1 单词1
 * @param word2 单词2
 * @returns 最少操作数
 */
function minDistance(word1: string, word2: string): number {
    const m = word1.length;
    const n = word2.length;
    
    // dp[j] 表示 word1[0..i-1] 转换成 word2[0..j-1] 的最少操作数
    const dp: number[] = Array.from({ length: n + 1 }, (_, i) => i);
    
    for (let i = 1; i <= m; i++) {
        let prev = dp[0];  // 保存 dp[i-1][j-1]
        dp[0] = i;         // 第一列
        
        for (let j = 1; j <= n; j++) {
            const temp = dp[j];  // 保存 dp[i-1][j]
            
            if (word1[i - 1] === word2[j - 1]) {
                dp[j] = prev;
            } else {
                dp[j] = Math.min(dp[j], dp[j - 1], prev) + 1;
            }
            
            prev = temp;
        }
    }
    
    return dp[n];
}

// 测试用例
console.log(minDistance("horse", "ros"));     // 3
console.log(minDistance("intention", "execution"));  // 5
console.log(minDistance("", ""));             // 0
console.log(minDistance("a", ""));            // 1
```

## 面试变体

1. **删除距离**：只能删除
2. **一次编辑**：判断是否可以通过一次编辑转换
3. **正则表达式匹配**：通配符匹配问题

## 常见错误

1. 一维优化时变量更新顺序错误
2. 初始化边界条件错误
3. 混淆行和列的索引
