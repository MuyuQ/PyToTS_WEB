## 学习建议

1. **按难度递进**：从 Easy 开始，逐步挑战 Medium 和 Hard
2. **先理解思路**：不要直接看代码，先自己思考解法
3. **对比实现**：注意 Python 和 TypeScript 在语法和类型系统上的差异
4. **面试准备**：重点关注「面试变体」和「常见错误」部分
