## 问题描述

一条包含字母 `A-Z` 的消息通过以下映射进行了**编码**：

```
'A' -> "1"
'B' -> "2"
...
'Z' -> "26"
```

要**解码**一条已编码的消息，所有数字必须分组，然后按上面的映射方法反向映射回字母（可能存在多种方式）。

例如，`"11106"` 可以映射为：
- `"AAJF"` ，将消息分组为 `(1 1 10 6)`
- `"KJF"` ，将消息分组为 `(11 10 6)`

注意，消息不能分组为 `(1 11 06)` ，因为 `"06"` 不能映射为 `"F"` ，这是由于 `"6"` 和 `"06"` 在映射中并不等价。

给你一个只含数字的**非空**字符串 `s`，请计算并返回**解码**方法的**总数**。

## 思路分析

动态规划：
- `dp[i]`：解码前 i 个字符的方法数
- 状态转移：
  - 单独解码 `s[i-1]`（如果是 '1'-'9'）：`dp[i] += dp[i-1]`
  - 和前一个字符组合解码（如果 `10 <= s[i-2:i] <= 26`）：`dp[i] += dp[i-2]`

## 复杂度分析

- **时间复杂度**：O(n)，遍历字符串
- **空间复杂度**：O(1)，只使用两个变量

## Python 实现

```python
def num_decodings(s: str) -> int:
    """
    解码方法 - Python 实现
    
    Args:
        s: 数字字符串
    
    Returns:
        解码方法数
    """
    if not s or s[0] == '0':
        return 0
    
    n = len(s)
    # dp[i] 表示前 i 个字符的解码方法数
    # 只保留前两个状态
    prev2 = 1  # dp[i-2]
    prev1 = 1  # dp[i-1]
    
    for i in range(1, n):
        current = 0
        
        # 单独解码当前字符
        if '1' <= s[i] <= '9':
            current += prev1
        
        # 和前一个字符组合解码
        two_digit = int(s[i-1:i+1])
        if 10 <= two_digit <= 26:
            current += prev2
        
        prev2, prev1 = prev1, current
    
    return prev1


# 测试用例
print(num_decodings("12"))    # 2 ("AB" 或 "L")
print(num_decodings("226"))   # 3 ("BBF", "BZ", "VF")
print(num_decodings("0"))     # 0
print(num_decodings("10"))    # 1 ("J")
print(num_decodings("27"))    # 1 ("BG")
```

## TypeScript 实现

```typescript
/**
 * 解码方法 - TypeScript 实现
 * @param s 数字字符串
 * @returns 解码方法数
 */
function numDecodings(s: string): number {
    if (s.length === 0 || s[0] === '0') {
        return 0;
    }
    
    const n = s.length;
    // 只保留前两个状态
    let prev2 = 1;  // dp[i-2]
    let prev1 = 1;  // dp[i-1]
    
    for (let i = 1; i < n; i++) {
        let current = 0;
        
        // 单独解码当前字符
        if (s[i] >= '1' && s[i] <= '9') {
            current += prev1;
        }
        
        // 和前一个字符组合解码
        const twoDigit = parseInt(s.substring(i - 1, i + 1));
        if (twoDigit >= 10 && twoDigit <= 26) {
            current += prev2;
        }
        
        prev2 = prev1;
        prev1 = current;
    }
    
    return prev1;
}

// 测试用例
console.log(numDecodings("12"));    // 2
console.log(numDecodings("226"));   // 3
console.log(numDecodings("0"));     // 0
console.log(numDecodings("10"));    // 1
console.log(numDecodings("27"));    // 1
```

## 面试变体

1. **带权解码**：每种解码方式有不同的权重
2. **限制解码**：某些组合不允许
3. **最长解码**：求最长解码字符串

## 常见错误

1. 忘记处理 '0' 的特殊情况
2. 两位数字范围判断错误（是 10-26 不是 1-26）
3. 忘记处理字符串以 '0' 开头的情况
