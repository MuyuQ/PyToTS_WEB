## 问题描述

你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1`。

在选修某些课程之前需要一些先修课程。先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [a, b]`，表示如果要学习课程 `a` 则必须先学习课程 `b`。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true`；否则，返回 `false`。

## 思路分析

拓扑排序：
- 构建有向图和入度数组
- 将所有入度为0的节点加入队列
- BFS遍历，每访问一个节点，将其邻居的入度减1
- 如果邻居入度变为0，加入队列
- 最后检查是否所有节点都被访问

## 复杂度分析

- **时间复杂度**：O(V + E)，V 是课程数，E 是先修课程数
- **空间复杂度**：O(V + E)，图和入度数组

## Python 实现

```python
from collections import deque

def can_finish(numCourses: int, prerequisites: list[list[int]]) -> bool:
    """
    课程表 - Python 实现 (拓扑排序)
    
    Args:
        numCourses: 课程总数
        prerequisites: 先修课程关系
    
    Returns:
        是否能完成所有课程
    """
    # 构建图和入度数组
    graph = [[] for _ in range(numCourses)]
    in_degree = [0] * numCourses
    
    for course, prereq in prerequisites:
        graph[prereq].append(course)
        in_degree[course] += 1
    
    # 将所有入度为0的课程加入队列
    queue = deque()
    for i in range(numCourses):
        if in_degree[i] == 0:
            queue.append(i)
    
    # 已学习的课程数
    learned = 0
    
    while queue:
        course = queue.popleft()
        learned += 1
        
        # 减少邻居的入度
        for next_course in graph[course]:
            in_degree[next_course] -= 1
            if in_degree[next_course] == 0:
                queue.append(next_course)
    
    return learned == numCourses


# 测试用例
print(can_finish(2, [[1, 0]]))          # True
print(can_finish(2, [[1, 0], [0, 1]]))  # False (循环依赖)
print(can_finish(4, [[1, 0], [2, 0], [3, 1], [3, 2]]))  # True
```

## TypeScript 实现

```typescript
/**
 * 课程表 - TypeScript 实现 (拓扑排序)
 * @param numCourses 课程总数
 * @param prerequisites 先修课程关系
 * @returns 是否能完成所有课程
 */
function canFinish(numCourses: number, prerequisites: number[][]): boolean {
    // 构建图和入度数组
    const graph: number[][] = Array.from({ length: numCourses }, () => []);
    const inDegree: number[] = new Array(numCourses).fill(0);
    
    for (const [course, prereq] of prerequisites) {
        graph[prereq].push(course);
        inDegree[course]++;
    }
    
    // 将所有入度为0的课程加入队列
    const queue: number[] = [];
    for (let i = 0; i < numCourses; i++) {
        if (inDegree[i] === 0) {
            queue.push(i);
        }
    }
    
    // 已学习的课程数
    let learned = 0;
    
    while (queue.length > 0) {
        const course = queue.shift()!;
        learned++;
        
        // 减少邻居的入度
        for (const nextCourse of graph[course]) {
            inDegree[nextCourse]--;
            if (inDegree[nextCourse] === 0) {
                queue.push(nextCourse);
            }
        }
    }
    
    return learned === numCourses;
}

// 测试用例
console.log(canFinish(2, [[1, 0]]));          // true
console.log(canFinish(2, [[1, 0], [0, 1]]));  // false
console.log(canFinish(4, [[1, 0], [2, 0], [3, 1], [3, 2]]));  // true
```

## 面试变体

1. **课程表 II**：返回一个可行的学习顺序
2. **并行课程**：最多同时上 k 门课，求最短时间
3. **最长路径**：求最长的先修课程链

## 常见错误

1. 图的方向搞反（应该是 prereq -> course）
2. 忘记处理入度数组
3. BFS结束后忘记判断是否所有节点都被访问
