## 问题描述

给定一个字符串 `s`，请你找出其中不含有重复字符的**最长子串**的长度。

## 思路分析

滑动窗口 + 哈希表：
- 使用左右指针维护一个窗口
- 用哈希表记录字符最近出现的位置
- 当遇到重复字符时，移动左指针到重复字符的下一个位置
- 更新最大长度

## 复杂度分析

- **时间复杂度**：O(n)，每个字符最多被访问两次
- **空间复杂度**：O(min(m, n))，m 是字符集大小

## Python 实现

```python
def length_of_longest_substring(s: str) -> int:
    """
    无重复字符的最长子串 - Python 实现
    
    Args:
        s: 输入字符串
    
    Returns:
        最长无重复子串的长度
    """
    char_index = {}  # 字符 -> 最近索引
    max_length = 0
    left = 0
    
    for right, char in enumerate(s):
        # 如果字符在窗口内，移动左指针
        if char in char_index and char_index[char] >= left:
            left = char_index[char] + 1
        
        # 更新字符位置
        char_index[char] = right
        
        # 更新最大长度
        max_length = max(max_length, right - left + 1)
    
    return max_length


# 测试用例
print(length_of_longest_substring("abcabcbb"))  # 3 ("abc")
print(length_of_longest_substring("bbbbb"))     # 1 ("b")
print(length_of_longest_substring("pwwkew"))    # 3 ("wke")
```

## TypeScript 实现

```typescript
/**
 * 无重复字符的最长子串 - TypeScript 实现
 * @param s 输入字符串
 * @returns 最长无重复子串的长度
 */
function lengthOfLongestSubstring(s: string): number {
    const charIndex = new Map<string, number>();  // 字符 -> 最近索引
    let maxLength = 0;
    let left = 0;
    
    for (let right = 0; right < s.length; right++) {
        const char = s[right];
        
        // 如果字符在窗口内，移动左指针
        if (charIndex.has(char) && charIndex.get(char)! >= left) {
            left = charIndex.get(char)! + 1;
        }
        
        // 更新字符位置
        charIndex.set(char, right);
        
        // 更新最大长度
        maxLength = Math.max(maxLength, right - left + 1);
    }
    
    return maxLength;
}

// 测试用例
console.log(lengthOfLongestSubstring("abcabcbb"));  // 3
console.log(lengthOfLongestSubstring("bbbbb"));     // 1
console.log(lengthOfLongestSubstring("pwwkew"));    // 3
```

## 面试变体

1. **最长无重复子串**：不只返回长度，还要返回子串本身
2. **至多k个不同字符**：允许最多 k 个不同字符的最长子串
3. **所有最长子串**：返回所有最长的无重复子串

## 常见错误

1. 忘记检查 `char_index[char] >= left`，导致左指针错误回退
2. 使用集合而不是哈希表，无法获取重复字符的位置
3. 更新最大长度的位置放错，导致结果错误
