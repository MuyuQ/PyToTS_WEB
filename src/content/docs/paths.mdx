# 学习路径

根据你的 Python 水平选择合适的学习路径。

## 路径概览

### 🌱 基础路径 (Foundation)

适合：Python 初学者，了解基本语法但缺乏前端经验

**目标**：掌握 TypeScript 核心语法和类型系统

**内容**：
- TypeScript 类型基础
- 变量与函数
- 接口与类型别名
- 泛型入门

### 🔄 迁移路径 (Migration)

适合：有 Python 后端经验的工程师

**目标**：建立 Python 到 TypeScript 的心智模型映射

**内容**：
- 类型系统对比
- 函数与参数
- 异步编程
- 模块与包管理

### 🚀 进阶路径 (Advanced)

适合：准备 TypeScript 面试或深入学习

**目标**：掌握工程实践和设计模式

**内容**：
- 高级类型技巧
- 设计模式实现
- 工程化最佳实践
- 面试题精讲
