# WebUI 重新设计文档

**日期:** 2026-04-14
**项目:** Python to TypeScript 双语学习站点
**设计方向:** Duolingo+GitHub 混合风格（教育温暖 + 技术极客）

---

## 1. 设计目标

### 1.1 目标受众

混合受众：学生群体（初学者）+ 在职开发者（有经验者）

### 1.2 设计原则

- **友好且专业** - 教育元素的温暖感 + 开发者的专业性
- **清晰层次** - 信息架构清晰，新手和进阶者都能高效使用
- **自动适应** - 根据系统偏好自动切换明暗主题
- **代码优先** - 双语代码块是核心，需要详细注释和良好可读性

### 1.3 成功标准

- 首屏加载时间 < 2s
- 颜色对比度符合 WCAG 2.1 AA 标准
- 移动端体验流畅，无横向滚动问题
- 代码块支持同步滚动和一键复制

## 4. 核心组件设计

### 4.1 双语代码块

**布局：**

- 左右分栏，Python 在左，TypeScript 在右
- 可折叠/展开任一语言
- 同步滚动
- 行号显示
- 一键复制（单个语言或双语）

**注释规范：**

- 每行关键逻辑都有注释（Python 用 `#`，TypeScript 用 `//`）
- 复杂算法步骤有分段注释块
- 变量命名保持一致
- 注释解释"为什么"而不仅是"做什么"
- 注释颜色与代码区分，确保可读性

**语法高亮：**

- 浅色：PrismJS GitHub 主题
- 深色：PrismJS Night Owl 主题

### 4.2 学习进度指示

**侧边栏状态：**

- ✓ 已完成（绿色）
- ◐ 进行中（橙色）
- ○ 未开始（灰色）

**进度显示：**

- 课程顶部显示进度条（百分比 + 视觉条）
- 完成测验后显示成就徽章
- 本地存储进度（localStorage）

### 4.3 测验组件

**设计：**

- 卡片式设计，圆角阴影
- 选项即时反馈（绿色正确/红色错误+解释）
- 可重做标记
- 支持单选和多选

**交互：**

- 点击选项即时显示对错
- 显示解释文字
- 完成后可重做
- 结果计入进度

### 4.4 导航组件

**左侧导航：**

- 分组折叠（学习路径/算法/手册/练习）
- 当前课程高亮（左侧蓝色竖线标记）
- 悬停显示子项
- 移动端：汉堡菜单，滑出式抽屉

**右侧目录：**

- 跟随滚动自动高亮当前章节
- 点击平滑滚动到对应位置
- 超过5个章节时显示滚动条

### 4.5 搜索组件

**功能：**

- 全局搜索（Cmd/Ctrl+K 快捷键）
- 结果按内容类型分组（课程/算法/手册）
- 高亮匹配关键词
- 支持模糊匹配

## 5. 响应式设计

### 5.1 断点

| 断点          | 布局        | 说明                     |
| ------------- | ----------- | ------------------------ |
| ≥1280px       | 完整三栏    | 导航+内容+目录           |
| 1024px-1279px | 双栏        | 隐藏右侧目录，内容区扩展 |
| 768px-1023px  | 单栏+可折叠 | 侧边导航可折叠           |
| <768px        | 移动端      | 汉堡菜单，全屏抽屉       |

### 5.2 移动端优化

- 底部固定导航栏（首页/搜索/目录）
- 代码块支持横向滑动
- 测验组件全宽按钮
- 字体大小适配（最小16px防止iOS缩放）
- 触摸目标最小44x44px

---

## 6. 性能优化

- 代码块懒加载（长页面按需渲染）
- 图片使用 WebP 格式
- 静态资源 CDN 缓存
- 首屏加载时间目标：<2s
- 使用 Astro Islands 架构，仅交互组件加载 JS

---

## 7. 无障碍访问

- 键盘导航支持（Tab/Enter/Escape）
- 屏幕阅读器友好（ARIA 标签）
- 颜色对比度符合 WCAG 2.1 AA 标准
- 减少动画选项（尊重 prefers-reduced-motion）
- 焦点指示器清晰可见
